💱 Currency Converter

A simple and responsive web application that converts currencies in real-time using a live exchange rate API.

🌐 Live Demo

👉 https://aryankumar62.github.io/Currency-converter/

🚀 Features

🌍 Convert between multiple currencies

🔄 Real-time exchange rates using API

🇺🇸🇮🇳 Dynamic country flags update

💡 Default conversion (USD → INR)

📱 Responsive and clean UI

🛠️ Tech Stack

HTML

CSS

JavaScript

Currency API (Exchange rate data)


⚙️ How It Works

Enter the amount

Select currencies (From / To)

Click Get Exchange Rate

See instant result with updated flags
